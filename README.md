# logic-box

## Overview

`logic-box` is a beginner-friendly Python command-line tool that demonstrates simple logic patterns:
- interactive menu-driven UI
- star-pattern generation
- even/odd range analysis with total sum

It is intended for learning Python control flow (`if`/`elif`/`else`), loops, and user input handling.

## Features

- Choice 1: generate star pattern up to given rows
- Choice 2: iterate range, print parity, compute sum
- Choice 3: exit cleanly

## Quick Start

```powershell
python logicbox.py
```

Then select:
- `1`: pattern generator
- `2`: number range analyzer
- `3`: exit

## Example

```
Select an option:
1. Generate a Pattern
2. Analyze a Range of Numbers
3. Exit
Enter your choice: 2

Enter the start of the range: 1
Enter the end of the range: 5

Number 1 is Odd
Number 2 is Even
Number 3 is Odd
Number 4 is Even
Number 5 is Odd

Sum of all numbers from 1 to 5 is: 15
```
